# 🌐 WebStudio Landing Page

## 📖 Описание
Профессиональный лендинг для веб-студии. Проект демонстрирует навыки создания современного многосекционного сайта с адаптивным дизайном, анимациями и продуманной структурой.

## 🛠️ Технологии
- **HTML5** — семантическая вёрстка
- **CSS3** — каскадные таблицы стилей
- **Flexbox** — построение гибких сеток
- **CSS Grid** — создание карточной структуры
- **Медиазапросы** — адаптивность под все устройства
- **CSS-переменные** — управление цветовой схемой
- **Font Awesome** — иконки (подключены через CDN)
- **Git** — контроль версий
